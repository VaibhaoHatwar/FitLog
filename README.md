# FitLog 🏋️ - MERN Stack Fitness Tracker

A full-stack fitness tracking app built with the MERN stack (MongoDB, Express, React, Node.js). Users can sign up, log in, and manage their workout logs with a clean UI and secure authentication system.

## 🛠 Tech Stack

- Frontend: React, Vite, Tailwind CSS (coming soon)
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Auth: JSON Web Tokens (JWT)
- Dev Tools: Git, VS Code, Postman

## 🔐 Environment Setup

Make sure to create a `.env` file inside your `backend` folder with the following content:

PORT=5000  
MONGO_URI=your_mongo_connection_string  
JWT_SECRET=your_secret_key
